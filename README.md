# AI Chatbot using Python 
This project is a simple AI chatbot built using Python and Scikit-Learn.

 ## Features - Takes user input - 
 Uses NLP text vectorization -
  Classifies user intent using Logistic Regression - 
  Returns meaningful responses - 
  Easy to extend with additional intents 
 
 ## Technologies Used 
 - Python -
  Scikit-Learn - 
  CountVectorizer - 
  Logistic Regression 
  ## How it Works 1. User enters a message. 
  2. Text is converted into numerical features. 
  3. Logistic Regression predicts the intent. 
  4. Chatbot returns a predefined response. 
  
  
  ## Run pip install -r requirements.txt python chatbot.py
